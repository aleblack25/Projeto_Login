# 🔐 Projeto Login

Uma tela de login moderna e responsiva desenvolvida com HTML e CSS, focada em design elegante, experiência do usuário e boas práticas de desenvolvimento Front-End.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Boxicons
* Google Fonts (Poppins)

## ✨ Funcionalidades

* Campo de usuário
* Campo de senha
* Opção "Lembrar senha"
* Link "Esqueci a senha"
* Botão de login com efeito hover
* Layout responsivo
* Design moderno com efeito glassmorphism
